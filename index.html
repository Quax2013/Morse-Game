Rules

Use the Straight Key, Space or Enter to send morse code using the Straight Key.

To send a "Dah" (the long sound) send a sound the length of 3 "Dit" (The short sound).

To end a letter pause for the lenght of 3 "Dit"

To end a word pause for the lenght of 7 "Dit".
